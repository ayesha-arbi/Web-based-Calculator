# 🔢 Modern Calculator Web App

A sleek and responsive calculator built using **HTML, CSS, and JavaScript**. This project focuses on delivering a clean user interface along with smooth functionality and keyboard support.

---

## 🚀 Features

* ✅ Basic arithmetic operations (+, −, ×, ÷)
* ✅ Responsive and modern UI design
* ✅ Keyboard input support
* ✅ Clear (AC) and Delete (DEL) functions
* ✅ Decimal support
* ✅ Smooth hover and click animations
* ✅ Error handling for invalid expressions

---

## 🖥️ Demo

Simply open the `index.html` file in your browser to run the app.

---

## 📂 Project Structure

```
calculator/
│── index.html    # Structure of the app
│── style.css     # Styling and layout
│── script.js     # Logic and functionality
```

---

## ⚙️ How It Works

* User inputs numbers and operators via buttons or keyboard
* Input is displayed on the screen in real-time
* JavaScript processes the expression using built-in evaluation
* Result is displayed instantly

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox & Grid)
* JavaScript (DOM Manipulation)
